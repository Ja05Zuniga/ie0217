Valida una dirección de correo electrónico según ciertas restricciones. 

La dirección de correo electrónico debe cumplir con las siguientes condiciones\+:
\begin{DoxyEnumerate}
\item El nombre puede contener letras, números, puntos (.) y guiones (-\/), pero no debe comenzar ni terminar con un punto o guion.
\item El nombre no debe tener más de 15 letras, números, puntos y guiones.
\item El dominio SOLO puede contener letras y debe tener al menos un punto al final.
\item La extensión puede contener solo letras y debe tener entre 2 y 4 caracteres.
\item No se permiten caracteres especiales adicionales en ninguna parte de la dirección de correo electrónico.
\end{DoxyEnumerate}
